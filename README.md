# My text based slot machine!!
Welcome to my new project😙

this is a text based slot machine game i made in python.Its not like real casino rules,i made my own rules!
If you bet on 1 line and win,you get 4x your bet.If you bet on 2 lines,you get 3x.And if you bet on all 3,you get 2x.Its all about risk vs reward!

## how to play
1. run the code
2. enter your diposit amount(must be a number)
3. chose how many lines to play(1,2 or 3).
4. enter your bet ammount.
5. cross your fingers and hope u win!!

## instalation
just download the code and run it in terminal.Make sure u have python installed.No special libraries needed,check requesments.txt just in case.

have fun playing!!
